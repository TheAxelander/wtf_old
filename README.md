# What's that function (WTF)

## Installation

```bash
curl https://raw.githubusercontent.com/TheAxelander/wtf/main/install.sh | bash
```

## Usage

```bash
wtf [[file]]
```

Example:

```bash
wtf docker
```

## Options:

```
-h,          Print script help
-t,          Run script in Dev mode (use files from current directory)
-u,          Update help files
-v,          Print script information
```
